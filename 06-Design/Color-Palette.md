# 🎨 Цветовая палитра

> Детальное описание цветов и их использование

---

## 🎨 Основная палитра (Brand Colors)

### Primary — Глубокий Байкал `#1e4a62`

```
HEX: #1e4a62
RGB: 30, 74, 98
HSL: 200°, 50%, 25%
CMYK: 69, 24, 0, 62
```

**Психология:** надёжность, глубина, доверие, экспертность

**Использование:**
- Заголовки H1-H6
- Логотип
- Основные кнопки (CTA)
- Ссылки
- Иконки важных элементов

**Контраст:** белый текст на этом фоне — AAA ✅

---

### Accent — Молния (Янтарь) `#f4b942`

```
HEX: #f4b942
RGB: 244, 185, 66
HSL: 41°, 89%, 61%
CMYK: 0, 24, 73, 4
```

**Психология:** энергия, тепло, действие, импульс

**Использование:**
- Кнопки CTA (вместо Primary, чтобы выделяться)
- Бейджи «Хит», «Новинка»
- Иконки уведомлений
- Акцентные элементы
- Hover-состояния

**Контраст:** тёмный текст на этом фоне — AAA ✅

---

### Secondary — Ледяной `#9fc2e8`

```
HEX: #9fc2e8
RGB: 159, 194, 232
HSL: 207°, 62%, 77%
CMYK: 32, 17, 0, 9
```

**Психология:** свежесть, лёд, чистота

**Использование:**
- Фоны секций
- Hover-состояния
- Подсветка
- Прелоадеры
- Лёгкие акценты

---

## ⚫ Нейтральные цвета

| Название | HEX | Использование |
|---|---|---|
| Угольный | `#1e2f3a` | Основной текст |
| Тёмно-серый | `#4d6e82` | Подписи, описания |
| Серый | `#6e8b9e` | Вторичный текст |
| Светло-серый | `#dce8ef` | Границы, разделители |
| Снежный | `#f8fafc` | Фон |
| Белый | `#ffffff` | Карточки, модалки |

---

## 🟢🟡🔴 Состояния

| Состояние | HEX | Использование |
|---|---|---|
| Success | `#22c55e` | В наличии, успешная оплата |
| Warning | `#eab308` | Под заказ, требуется внимание |
| Danger | `#ef4444` | Нет в наличии, ошибки |
| Info | `#3b82f6` | Уведомления |

## 📊 Контрастность (WCAG)

| Комбинация | Контраст | Уровень |
|---|---|---|
| `#1e4a62` на белом | 9.8:1 | AAA ✅ |
| `#1e4a62` на `#9fc2e8` | 4.6:1 | AA ✅ |
| `#f4b942` на `#1e2f3a` | 8.5:1 | AAA ✅ |
| `#1e2f3a` на белом | 14.5:1 | AAA ✅ |
| `#4d6e82` на белом | 6.2:1 | AAA ✅ |
| `#f4b942` на белом | 2.4:1 | ❌ FAIL (только крупно) |
