# 🛍 Карточка товара — макет

> Структура новой карточки (вдохновлена SkatePRO)

---

## 🎯 Цели карточки

1. Показать товар со всех сторон
2. Объяснить, для кого и зачем
3. Убедить в качестве
4. Снять возражения (FAQ, отзывы)
5. Упростить покупку

---

## 📐 Структура

### 1️⃣ Хлебные крошки
```
Главная > Готовые коньки > Лезвие «ГРОМ» неокрашенное
```

---

### 2️⃣ Основной блок (две колонки)

#### Левая колонка: Галерея (60%)

**Элементы:**
- Главное фото (high-res)
- Миниатюры (5-7 фото)
- Видео (1-2)
- 360° обзор (опционально)
- Зум при наведении

**Макет:**
```
┌──────────────────────────┐
│                          │
│                          │
│      [Главное фото]      │
│                          │
│                          │
└──────────────────────────┘
[1][2][3][4][▶️][360°]
```

#### Правая колонка: Информация (40%)

**Элементы (sticky при скролле):**
- Бейдж «Хит» (если есть)
- H1: Название товара
- Рейтинг (звёзды + количество отзывов)
- Цена (крупно)
- Старая цена + скидка (если акция)
- Краткое описание (2-3 строки)
- Кнопка "Добавить в избранное"
- Выбор вариации (размер, цвет)
- Калькулятор размера (ссылка)
- Количество (input)
- Кнопка «В корзину» (primary, large)
- Кнопка «Купить в 1 клик» (secondary)
- Trust-элементы:
  - 🚚 Доставка за 3-7 дней
  - 💯 Гарантия 1 год
  - 🔄 Возврат 14 дней
  - 📞 Поддержка в MAX
- Иконки оплаты (СБП, карты)

**Макет:**
```
[Бейдж: Хит]

Лезвие «ГРОМ» неокрашенное

⭐⭐⭐⭐⭐ 4.8 (12 отзывов)

6 600 ₽

Стальное лезвие для открытого льда.
Подходит под лыжные ботинки NNN и SNS.

[❤ Избранное]

Размер: [▼ 480 мм] [Не знаете размер?]

Количество: [- 1 +]

[В корзину]
[Купить в 1 клик]

🚚 Доставка 3-7 дней
💯 Гарантия 1 год
🔄 Возврат 14 дней
📞 Поддержка в MAX

[СБП] [Карты] [Robokassa]
```

## 📱 Мобильная версия

### Изменения
- Галерея: полноэкранный слайдер
- Sticky-кнопка: всегда видна внизу
- Табы вместо длинной страницы (или скролл)
- Калькулятор размера — отдельный экран
- Видео — fullscreen
- Отзывы — горизонтальный скролл

---

## 🎨 Визуальные акценты

- Главное фото — большое, детальное
- Trust-элементы — иконки + текст
- Sticky-кнопка — выделяется на любом фоне
- Бейджи — заметные, но не агрессивные
