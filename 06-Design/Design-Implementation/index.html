    🎨 Реализация: «ГРОМ и Молния» (Bold Industrial)¶

    Дата: 27.06.2026 Статус: ✅ Реализовано в Astro, v2 — адаптивность + i18n Скриншоты: /home/za/Desktop/Grom/GromProject/screenshot-fullpage.png (1910×5113, полная страница)

    🔄 Обновление v2: Адаптивность, i18n, читаемость карточек¶

    Дата: 27.06.2026

    Исправленные проблемы¶

    1. Адаптивность — при уменьшении ширины всё смещалось вправо
    2. ✅ Добавлен overflow-x-hidden на body
    3. ✅ Заголовки: text-4xl sm:text-6xl lg:text-8xl (вместо text-5xl)
    4. ✅ Grid gaps: gap-3 sm:gap-6 (вместо gap-4/gap-6)
    5. ✅ Кнопки: text-sm sm:text-base
    6. ✅ Header уменьшен до h-14 sm:h-16
    7. ✅ Tagline скрыт на мобильных (hidden sm:block)
    8. ✅ Калькулятор: grid sm:grid-cols-2

    9. ✅ Theme-switcher кнопки: 24×22px на мобильных, 28×26px на sm+

    10. Читаемость карточек товаров

    11. ✅ text-frost-dark/60 → text-frost-dark/70 (более читаемо)
    12. ✅ text-frost-dark/50 → text-frost-dark/60 для SKU
    13. ✅ Цена: text-xl sm:text-2xl (вместо text-2xl)

    14. ✅ Описание: text-xs sm:text-sm

    15. Русификация

    16. ✅ «SKU · » → «Арт. »
    17. ✅ «GROM-001» → «ГРОМ-001»
    18. ✅ «ШЕСТЬ SKU. ОДИН СТАНДАРТ.» → «ШЕСТЬ ИЗДЕЛИЙ. ЕДИНЫЙ СТАНДАРТ.»

    19. ✅ «ДЛИТЕЛЬНОСТЬ · {days} ДНЕЙ» → «Длительность · {days} дн.»

    20. i18n (интернационализация)

    21. ✅ Создан src/i18n/translations.ts с 3 языками (ru/en/zh)
    22. ✅ Переключатель языков в header (🇷🇺 🇬🇧 🇨🇳)
    23. ✅ Layout использует ?lang=ru|en|zh query parameter
    24. ✅ <html lang> обновляется динамически

    25. ✅ Мета-теги и footer обновлены через словари

    26. Темы (3 уникальных)

    27. ✅ 🌙 Байкальская ночь — deep navy + icy blue (НЕ жёлтый!)
    28. ✅ 🌅 Байкальский рассвет — frost + soft amber (default)
    29. ✅ 🪵 Таёжный закат — warm paper + copper
    30. ✅ --muted на night увеличен до #94aec6 для лучшей читаемости

    Переключатель языков¶

    Добавлен в header (рядом с переключателем тем): 

    <div class="theme-switcher" id="lang-switcher">
  <a href="?lang=ru" class="theme-btn active">🇷🇺</a>
  <a href="?lang=en" class="theme-btn">🇬🇧</a>
  <a href="?lang=zh" class="theme-btn">🇨🇳</a>
</div>

    Параметр: ?lang=ru|en|zh Default: ru Словарь: src/i18n/translations.ts (70+ ключей) Применение: const tr = t[lang] в layout

    Файлы¶

    • src/i18n/translations.ts — словари (ru/en/zh)
    • src/layouts/Base.astro — layout с i18n и темами
    • src/pages/index.astro — главная (русифицирована)
    • src/pages/catalog.astro — каталог (русифицирован)
    • src/pages/product/[sku].astro — карточка товара
    • src/pages/tours/index.astro — каталог туров
    • src/pages/tours/[slug].astro — страница тура
    • src/pages/technology.astro — технология
    • src/pages/about.astro — о бренде
    • src/pages/contacts.astro — контакты
    • src/styles/global.css — стили с CSS-переменными

    Что увидит пользователь¶

    • При первом заходе: Байкальский рассвет (светлая) + Русский
    • При клике на 🌙: deep navy + icy blue акценты + читаемые заголовки
    • При клике на 🇬🇧: интерфейс переключается на English
    • При сужении браузера: всё перестраивается корректно, нет горизонтального overflow

    ✅ Что реализовано¶

    1. Палитра (фирменные цвета)¶

    Имя HEX Использование
    thunder #0a1929 Грозовая синь — основной тёмный
    ice #2c5a6e Байкальский лёд — акцент
    bolt #f4b942 Молния — главный CTA-цвет
    frost #e8f1f7 Снег — фон
    frost-dark #c9d9e6 Границы
    dawn #9fc2e8 Рассвет
    steel #1a1a1a Уголь — текст
    carbon #2a2a2a Тёмный фон

    2. Типографика¶

    • Заголовки: Inter (700-900) + UPPERCASE + industrial letter-spacing (0.12em)
    • Текст: Inter (400-500)
    • Акценты (SKU, цены, телефоны): JetBrains Mono

    3. Уникальные визуальные элементы¶

    ⚡ Логотип-молния (SVG inline)¶

    • Главный графический символ
    • Используется в header, hero, footer, бейджах
    • Стилизованный path (5 точек, угловатый)

    🧊 Текстура льда (.bg-ice-texture)¶

    • Тёмный фон с радиальными градиентами (ice + bolt)
    • SVG-noise наложение (feTurbulence)
    • Эффект глубины и замерзания

    🛼 След лезвия (.skate-track)¶

    • Анимированный паттерн (бесконечная прокрутка)
    • SVG-path с волнистой линией
    • Цвет: bolt-yellow с opacity 0.4

    ❄️ Трещины льда (.ice-cracks-bg)¶

    • Многослойные изогнутые SVG-линии
    • Создают текстуру замёрзшего озера
    • На светлых фонах

    💥 Разделитель-молния (.divider-bolt)¶

    • Горизонтальная линия с ⚡-иконкой по центру
    • Светящийся круг вокруг молнии
    • Используется в footer

    🔧 Карточка товара (.product-card)¶

    • Скошенный уголок-уголок (clip-path) с ⚡-значком
    • Hover: подъём + 8px жёлтая тень
    • Косая clip-path на кнопках (как у тракторов)

    4. Структура страницы¶

    Секция Содержание
    Header Sticky, тёмно-синий, логотип-молния + навигация в industrial-стиле + телефон
    Hero Тёмный фон, координаты Ангарска, 3-уровневый заголовок, кнопки, метрики (109/700/110), плашка продукта GROM-001
    Технология Светлый фон, 4 технические характеристики (Сталь 420, Алюминий, Размеры, Нагрузка)
    Отзывы Тёмный фон, 3 реальных отзыва с городами, рейтинг 5★
    Каталог Светлый фон, 6 товаров с SKU, ценами, бейджами "ХИТ"
    Калькулятор Тёмный фон, два select (размер EU + тип крепления), рекомендация
    CTA Светлый фон, ⚡-иконка, призыв к действию
    Footer Тёмно-синий, .skate-track разделитель, 4 колонки, мигающий индикатор

    5. Tone of voice (визуальный)¶

    • ❌ Нет «добро пожаловать в наш магазин»
    • ✅ «ОЗЁРНЫЕ КОНЬКИ ГРОМ»
    • ✅ «НЕ ГНЁТСЯ. НЕ ЛОМАЕТСЯ.»
    • ✅ «ШЕСТЬ SKU. ОДИН СТАНДАРТ.»
    • ✅ «ПОДОБРАТЬ ЛЕЗВИЕ»
    • ✅ «ГОТОВ К ОТКРЫТОМУ ЛЬДУ?»
    • ✅ «Сделано в Ангарске · Испытано на Байкале»

    6. Уникальность на рынке¶

    До: generic e-commerce на Tailwind (как у всех)

    После: - ⚡ Геокоординаты Ангарска в hero (N 52°31′ · E 103°54′) - 🧊 Текстура замёрзшего озера (SVG noise) - 🛼 Анимированный след лезвия в hero и footer - 🏭 Industrial-стиль (UPPERCASE, mono, скошенные углы) - 📍 «EST. 2018 · ООО «ГРОМ»» — копания в бренде - 🔩 Технические характеристики в стиле datasheet (Артикул, Сталь, Длина, Вес, Ресурс) - 💡 «700 км без заточки» как главный hero-блок - ⚙️ Цифры в industrial-стиле (109 отзывов · 5.0, 700 км, 110 кг)

    📊 Сравнение «до» и «после»¶

    Критерий До (default) После (ГРОМ и Молния)
    Палитра Generic синий+жёлтый Грозовая синь + байкальский лёд + молния
    Логотип Эмодзи ⛸ SVG-молния
    Hero Сине-голубой градиент Тёмный с текстурой льда + спецификация
    Заголовки Sentence case UPPERCASE + industrial letter-spacing
    Карточки Белый фон + тень Скошенный ⚡-уголок + жёлтая тень
    Кнопки Скруглённые Скошенные (clip-path)
    Отзывы Нет 3 реальных с городами
    Калькулятор Нет Интерактивный с select
    Гео-привязка Нет Координаты Ангарска, локации в отзывах
    Tone of voice Корпоративный «НЕ ГНЁТСЯ. НЕ ЛОМАЕТСЯ.»
    Запоминаемость Низкая (как у всех) Высокая (молния + сибирский характер)

    🔧 Технические детали¶

    Tailwind конфиг (tailwind.config.mjs)¶

    colors: {
  thunder: { DEFAULT: '#0a1929', light: '#1a2f48', dark: '#050d18' },
  ice: { DEFAULT: '#2c5a6e', dark: '#1a3a4e', light: '#5b8db8' },
  bolt: { DEFAULT: '#f4b942', dark: '#e0a830', light: '#ffc857' },
  frost: { DEFAULT: '#e8f1f7', dark: '#c9d9e6' },
  dawn: '#9fc2e8',
  steel: '#1a1a1a',
  carbon: '#2a2a2a',
}

letterSpacing: {
  'industrial': '0.12em',
  'wider-2': '0.18em',
}

animation: {
  'bolt-pulse': 'boltPulse 2s ease-in-out infinite',
  'frost-shift': 'frostShift 8s ease-in-out infinite',
  'skate-roll': 'skateRoll 8s linear infinite',
}

    CSS-компоненты (global.css)¶

    • .bg-ice-texture — тёмный фон с текстурой замёрзшего озера
    • .skate-track — анимированный паттерн следа лезвия
    • .ice-cracks-bg — паттерн трещин на светлом фоне
    • .divider-bolt — разделитель с ⚡-иконкой
    • .btn-bolt, .btn-steel, .btn-ghost — кнопки в industrial-стиле
    • .badge-bolt, .badge-steel — скошенные бейджи
    • .product-card — карточка с ⚡-уголком

    Использованные данные (реальные)¶

    • 6 SKU из эпистемического аудита
    • 109 отзывов (5.00/5)
    • 3 реальных отзыва с именами и городами
    • Реальные характеристики (сталь 420, 460/520мм, 800/880г, ≥200км, до 110кг)
    • Цены (1 200-9 200 ₽)
    • Контакты ООО «ГРОМ» (ИНН 3801165352, Ангарск)
