# 🔤 Типографика

> Шрифты, размеры, веса, использование

---

## 🎯 Шрифты

### UI / Body: **Inter**
```
Семейство: Inter
Стили: Regular 400, Medium 500, SemiBold 600, Bold 700
Файл: локально (без Google Fonts)
Замена: Manrope, Onest
```

**Почему Inter:**
- ✅ Бесплатный
- ✅ Создан для UI
- ✅ Множество весов
- ✅ Отличная читаемость на экранах
- ✅ Поддержка кириллицы
- ✅ Variable font

**Где используем:**
- Основной текст
- Кнопки
- Формы
- Навигация
- Подписи

---

### Headings: **TT Norms Pro**
```
Семейство: TT Norms Pro
Стили: Regular 400, Medium 500, Bold 700, Black 900
Файл: лицензия (платный, есть русская)
Замена: Involve, Golos Text
```

**Почему TT Norms Pro:**
- ✅ Качественная кириллица
- ✅ Создан в РФ
- ✅ Характерный, но не агрессивный
- ✅ Множество весов

**Где используем:**
- Заголовки H1-H6
- Hero-текст
- Подзаголовки

---

### Accent: **Unbounded**
```
Семейство: Unbounded
Стили: Bold 700, ExtraBold 800, Black 900
Файл: бесплатный (Google Fonts, но лучше локально)
Замена: Russo One
```

**Почему Unbounded:**
- ✅ Характерный геометрический шрифт
- ✅ Поддержка кириллицы
- ✅ Бесплатный
- ✅ Сильный «голос»

**Где используем:**
- Hero-баннеры
- Лозунги
- Акцентные надписи

| Вес | Название | Использование |
|---|---|---|
| 400 | Regular | Основной текст |
| 500 | Medium | Кнопки, навигация, подписи |
| 600 | SemiBold | Подзаголовки, акценты |

| 800 | ExtraBold | Hero |

## 📐 Правила типографики

### Line-height
- Заголовки: 1.1-1.2
- Основной текст: 1.5-1.6
- Подписи: 1.4

### Letter-spacing
- Заголовки: -0.02em
- Основной текст: 0
- Подписи caps: 0.05em (расширить)

### Длина строки
- Оптимально: 60-75 символов
- Минимум: 40 символов
- Максимум: 90 символов

### Параграфы
- Отступ между параграфами: 1em
- Отступ первой строки: 0 (не используем)
- Выравнивание: left (не justify)

## 📝 Использование в компонентах

### Кнопка
```
Шрифт: Inter, Medium 500, 16px
Высота: 48px (large), 40px (default), 32px (small)
Цвет текста: #ffffff или #1e2f3a
```

### Форма
```
Label: Inter Medium 500, 14px, #4d6e82
Input: Inter Regular 400, 16px, #1e2f3a
Placeholder: Inter Regular 400, 16px, #6e8b9e
```

### Карточка товара
```
Заголовок: TT Norms Pro Bold 700, 18px, #1e4a62
Цена: Inter SemiBold 600, 20px, #1e2f3a
Подпись: Inter Regular 400, 14px, #4d6e82
```

### Навигация
```
Меню: Inter Medium 500, 16px, #1e2f3a
Активный: Inter SemiBold 600, #1e4a62
```
