# 📝 Контент-аудит

> Описания товаров, тексты, медиа, копирайт

**Общая оценка: 1/10** — практически весь контент отсутствует.

---

## 📄 Текстовый контент

### Главная страница
- 🔴 **0 символов текста** — только картинка + 2 кнопки
- Нет УТП
- Нет описания бренда
- Нет преимуществ
- Нет призыва к действию

### Карточки товаров (6 шт)
- 🔴 **Все 6 short_description пустые**
- 🔴 **Все 6 description пустые**
- Только заголовок + цена + характеристики (вес, размеры)

### Каталог
- 🟡 Только название категории «Гром»
- Нет описания категории

### Страницы
- 🟡 «Контакты» — только реквизиты, нет описания
- 🟡 «Магазин спортивного инвентаря» — картинки без текста
- 🔴 «В разработке» — заглушка, должна быть `noindex`

### Блог
- 🔴 **Блога нет вообще**

## 🛍 Товары — что должно быть

### Карточка «Лезвие "ГРОМ" неокрашенное»
**Текущее:**
- Заголовок: «Лезвие "ГРОМ" неокрашенное»
- Цена: 6 600 ₽
- Характеристики: Вес 1,1 кг, 57×13×7 см
- Артикул: 003
- Размеры: 480, 520

**Должно быть:**
- 📝 Описание (300-500 слов):
  - Для чего: открытый лёд, байкальский лёд, замёрзшие озёра
  - Материал: какая сталь, твёрдость
  - Заточка: фабричная / требует доработки
  - Совместимость: какие крепления (NNN, SNS)
  - Для каких ботинок (размерный ряд)
  - Уход: как точить, как хранить
- 📊 Характеристики (таблица):
  - Сталь
  - Длина лезвия
  - Ширина
  - Толщина
  - Вес
  - Твёрдость (HRC)
  - Совместимость с ботинками
  - Геометрия заточки
- 📸 Фотогалерея (5-10 фото):
  - Лезвие в упаковке
  - Вид сверху, сбоку, снизу
  - С креплением
  - На ботинке
  - В сравнении с обычным коньком
  - С линейкой
- 🎬 Видео (1-2 шт):
  - Процесс катания
  - Установка на ботинок
- ⭐ Отзывы (цель: 10+ с фото)
- ❓ FAQ (3-5 вопросов)
- 🔗 «С этим товаром покупают» (cross-sell)

## 📸 Медиа

### Фото
- **Hero:** gr.webp (1376×768, 173 КБ) ✅ приемлемо
- **Логотип:** gr-1.webp (200×182, 28 КБ) ⚠️ нужен SVG
- **Товарные:** JPG 1-2 МБ, ресайзы есть ✅
- **Портфолио/кейсы:** ❌ нет
- **Lifestyle:** ❌ нет

### Видео
- ❌ На сайте нет ни одного видео
- Потенциал: обзоры на YouTube/RuTube, embed на карточки

### UGC (User Generated Content)
- ❌ Не используется
- Потенциал: отзывы с фото из MAX/ВК/Telegram

## 🏷 Копирайт и tone of voice

### Что есть
- Минимальный, технический
- Названия товаров: «Лезвие "ГРОМ" неокрашенное» — сухо
- Кнопки: «Выбрать параметры», «В корзину» — стандарт

### Чего не хватает
- Бренд-голос
- Эмоциональный контент (истории клиентов)
- Образовательный контент (как выбрать, как ухаживать)
- SEO-тексты в категориях
