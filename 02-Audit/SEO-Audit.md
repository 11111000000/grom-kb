## 🏷 Мета-теги

### Title (Title)

| Страница | Title | Проблема |
|---|---|---|
| Главная | «ГРОМ38 озерные коньки» | OK, но слабый |
| Лезвие неокрашенное | «Гром Лезвие «ГРОМ» неокрашенное» | Дублирование бренда |
| Каталог | «Гром - ГРОМ» | Дублирование |
| Контакты | «Контакты - ГРОМ» | OK |
| Магазин | «Магазин спортивного инвентаря - ГРОМ» | OK |
| Разработка | «Страница в разработке - ГРОМ» | Нет смысла индексировать |
| Оферта | «Публичная оферта - ГРОМ» | OK |

**Проблемы:**
- 🔴 Нет ключевых слов в title для категорий
- 🔴 Длина title > 60 символов у товаров
- 🟡 Не используются модификаторы («купить», «цена», «доставка»)

### Description (Meta Description)

**🔴 ПОЛНОСТЬЮ ОТКЛЮЧЕН ТЕМОЙ** — комментарий в коде: `<!-- meta description disabled by the theme -->`

**Результат:** Яндекс и Google сами формируют сниппет, часто неудачно.

### OG-description (соцсети)

**Спам ключевыми словами:**
- Главная: «озерные коньки ГРОМ озерные коньки гром озёрные коньки гром гром38»
- Карточка: «Лезвие "ГРОМ"» (1 слово)
- Контакты: 2000+ символов реквизитов в description

**🔴 Это явный анти-SEO паттерн.**

## 🧬 Schema.org (микроразметка)

### Что есть
- ✅ Organization
- ✅ WebSite
- ✅ BreadcrumbList
- ✅ Product (на карточках)
- ✅ Offer с price, availability, priceCurrency
- ✅ Brand

### Критические ошибки
1. 🔴 `itemCondition: "UsedCondition"` для **новых** товаров
2. 🔴 `Offer.url` дублируется (3 раза)
3. 🔴 `Offer.seller.name = "ГРОМ"` — должен быть полным «ООО ГРОМ»
4. 🔴 `publisher.telephone = "ГРОМ"` (плейсхолдер, не число)
5. 🟡 Нет `Product.description` (пустая строка)
6. 🟡 Нет `AggregateRating` (0 отзывов)
7. 🟡 Нет `image.caption` для товаров
8. 🟡 `priceValidUntil: "2027-12-31"` — слишком далеко

## 🎯 SEO-план

### P0 (срочно)
1. Включить meta description
2. Убрать спам keywords из OG-description
3. Заполнить alt-ы у всех изображений
4. Исправить Schema.org (`NewCondition` вместо `Used`)
5. Добавить H1 на главную

### P1 (в течение месяца)
6. Переписать title и description для всех страниц
7. Создать HTML-карту сайта
8. Добавить блог с 5+ SEO-статьями
9. Внутренняя перелинковка
10. Настроить `AggregateRating`

### P2 (развитие)
11. Сбор семантики в Key Collector
12. Оптимизация под LSI-слова
13. Турбо-страницы / Я.Товары
14. Schema.org `FAQPage`, `HowTo`
