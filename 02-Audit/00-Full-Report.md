## 4. ДИАЛЕКТИЧЕСКИЙ АНАЛИЗ (тезис → антитезис → синтез)

### 4.1 Главная страница

**Тезис (что есть):** Минимализм — большая шапка-картинка + две кнопки. Быстро, понятно куда идти.

**Антитезис (проблемы):**
- Картинка-герой — это просто PNG/JPG с буквами «ГРОМ» (статика, не рассказывает историю)
- Нет **позиционирования бренда** — кто вы? почему именно «ГРОМ»? что особенного?
- Нет **социального доказательства** — отзывов, цифр, кейсов, фото с Байкала
- Нет **hero-CTA с ценностью** — кнопки просто «куда-то ведут»
- 0 текстового контента = плохо для SEO (Яндекс не понимает тематику)
- OG-description = «озерные коньки гром озёрные коньки гром гром38» — спам ключами

**Синтез (что делать):**
- Главная — это **витрина бренда + продуктовый лендинг**. Нужны: УТП, 2-3 блока-преимущества, витрина хитов, отзывы, локация, FAQ
- Hero-видео или кинематографичные фото катания по Байкалу
- Подзаголовок: «Коньки для открытого льда. Сделано в Ангарске. Испытано на Байкале.»

### 4.2 Навигация

**Тезис:** Шапка — логотип, телефон, вход, корзина, избранное, гамбургер. Меню: «Магазин» + «Контакты».

**Антитезис:**
- Меню из 2 пунктов — это **отсутствие навигации**. Где «О нас», «Доставка/оплата», «Гарантия», «Блог», «Как выбрать размер»?
- В каталоге **нет ни одного фильтра** — только сортировка
- **Нет категорий второго уровня** (лезвия / крепления / аксессуары / готовые коньки)
- Меню не адаптировано под два бизнеса — турагентство спрятано в кнопке
- **Несколько разных телефонов**: в шапке `+7 (901) 6414848`, на странице контактов `+7 (902) 5761917`. Два номера в одном бизнесе — путаница

**Синтез:**
- Полноценная навигация: каталог с категориями, о бренде, доставка, контакты, блог
- Подкатегории каталога: Готовые коньки / Лезвия / Крепления / Аксессуары / Кастом
- Фасетный фильтр: по размеру, цене, цвету, типу
- Один телефон, читаемая шапка, sticky-хедер

### 4.3 Карточка товара

**Тезис:** Стандартная WC-карточка с галереей, ценой, выбором вариации, кнопкой «В корзину», вкладками «Детали/Отзывы».

**Антитезис:**
- **Критично:** у товаров **нет описания** — `short_description` пуст, `description` пуст
- Нет характеристик: материал лезвия, тип стали, заточка, назначение, для какого льда
- У товара «Лезвие окрашенное» и «неокрашенное» — одинаковые фото? Нужно сравнение
- Schema.org `itemCondition: UsedCondition` для нового товара — **ошибка** (SEO)
- Нет блока «С этим товаром покупают» / cross-sell (лезвие + крепление = комплект)
- **Нет отзывов** — 0 на всех товарах (блок выглядит мёртвым)
- Robokassa-виджет огромный, вытесняет основной контент
- Нет видео-обзора, нет размерной сетки, нет «как подобрать размер под ботинок»
- `SKU: 003, 004, 005, 006, 001, 002` — хаотичная нумерация

**Синтез:**
- Полноценное описание: 2-3 абзаца текста + буллеты + характеристики
- Блок-схема: «Под ботинок NNN 38-46 → лезвие 480/520»
- Видео: лезвие в деле на байкальском льду
- Галерея с зумом, видео, 360°
- Sticky-блок «купить» при скролле
- Комплекты: лезвие + крепление + чехол = −15%
- Калькулятор размера

### 4.4 Каталог

**Тезис:** Сетка 5 колонок, hover-эффект, бейджи «Хит!».

**Антитезис:**
- 5 колонок на десктопе = **мелкие карточки**, плохая читаемость
- Фото с разным соотношением сторон — кривая сетка
- Нет категорий, нет фильтров, нет пагинации (т.к. 6 товаров)
- Названия типа «Мехaничеcкое кpeплeниe» — **явная кириллица в латинице** (XSS через иммитацию), вредно для SEO и бренда
- Alt-ы у фото — `IMG_20251128_155632_918` (системные имена), не индексируется
- Бейджи «Хит!» на 3 товарах из 6 — обесценивает бейдж
- Кнопка «Выбрать параметры» на variable-товарах (а не «В корзину») — лишний клик

**Синтез:**
- 3-4 колонки на десктопе, 2 на планшете, 1 на мобиле
- Единые пропорции фото 1:1 или 4:3
- Альт-ы осмысленные: «Озёрные коньки ГРОМ неокрашенные, вид сбоку»
- Quick-view с AJAX
- Фильтры в сайдбаре

### 4.5 Контакты

**Тезис:** Карта + реквизиты ООО + адрес.

**Антитезис:**
- **Два разных телефона** в одном бизнесе — путаница
- Email `qrom.max38t@vandex.ru` — **опечатка «vandex» вместо «yandex»** (также странный домен)
- Реквизиты Т-банка + юрадрес в квартире — выглядит не солидно для производства
- Нет часов работы, нет формы обратной связи, нет WhatsApp/MAX кнопки (хотя MAX уже есть как плавающая)
- Нет схемы проезда, парковки, что рядом

**Синтез:**
- Один телефон, один email с человеческим доменом (info@grom38.ru)
- Отдельная страница «Реквизиты» для бухгалтеров
- Часы работы, форма «Задать вопрос», MAX/WhatsApp/Telegram

### 4.6 Дизайн и визуал

**Тезис:** Светлая тема bono, шрифт-иконки, скруглённые элементы.

**Антитезис:**
- **Логотип — растровый webp 200×182**, не векторный SVG. На retina — мыло
- Палитра: по коду страницы «в разработке» — `#1e4a62` (тёмно-синий) + `#f4b942` (жёлтый) + `#eef4f8` (фон). Но на основном сайте эта палитра **не используется** — там дефолтные цвета темы bono (серо-бело-синий без характера)
- **Нет брендинга**: уникальности «ГРОМ» не видно — выглядит как стандартный WC-магазин
- Hero-картинка — скучная композиция с буквой «Г»
- Нет lifestyle-фотографий (люди на льду, Байкал, эмоции)
- Cookie-бар с устаревшим текстом и ссылкой «Ok» — выглядит как спам
- Не используется видео вообще

**Синтез:**
- Векторный логотип SVG, фирменный знак (например, стилизованное лезвие + молния)
- Брендовая палитра: глубокий синий Байкала + ледяной голубой + акцентный оранжевый (или янтарный)
- Типографика: гротеск для UI + акцидентный шрифт для заголовков
- Hero — видео/фото катания по льду Байкала
- Lifestyle-блок: фото в деле
- Айдентика: паттерн следа лезвия на льду

### 4.7 Контент-стратегия

**Тезис:** Каталог + контакты + оферта + одна заглушка.

**Антитезис:**
- **Ноль информационного контента**: нет блога, нет статей «Как выбрать озёрные коньки», «Как кататься по открытому льду», «ТОП-5 мест на Байкале зимой»
- OG-description везде дублирует keywords
- H1 = название товара, но нет подзаголовков H2/H3 с УТП
- Изображения загружаются с системными именами (IMG_20251128_155632_918.jpg) — никто их не гуглит
- Нет видео-обзоров, отзывов с фото, UGC
- Email в форме регистрации требует верификацию (Customer Email Verification) — снижает конверсию регистрации

**Синтез:**
- Блог: «Как выбрать размер лезвия», «Гайд по катанию на байсах», «Байкал зимой: лучшие маршруты», «Уход за лезвием»
- SEO-структура: категории + теги + статьи
- Видео-обзоры на YouTube/RuTube → embed
- UGC из ВК/MAX/Telegram

### 4.8 Технические баги

- **Schema.org ошибка**: `itemCondition: "UsedCondition"` для нового товара → Google может пессимизировать
- **Meta description отключён темой** → пустой сниппет в поиске
- Валидация форм **отключена JS-патчем** — серьёзная проблема для сбора лидов
- Дублирование кнопки MAX (плавающая вставлена дважды — на body и в footer)
- Cookie-баннер — спам-текст: «Caйт иcпoльзуeт куки-фaйлы, чтoбы cдeлaть вaшe пpeбывaниe нa нeм мaкcимaльнo удoбным» (замена русских букв на латиницу — антиспам-обход)
- `variation_id=0` при отсутствии выбора — UX-проблема
- `<title>` у товара = «Гром Лезвие «ГРОМ» неокрашенное» — дублирование бренда
- Schema `publisher` без реального `telephone` (`"telephone":"ГРОМ"` — placeholder)
- `mailto:qrom.max38t@vandex.ru` — опечатка в домене

## 7. ИТОГОВЫЙ ВЕРДИКТ

Сайт **технически функционален, но маркетингово пустой**. Это типичный «магазин в разработке»: товары заведены, оплата подключена, но нет ни описаний, ни отзывов, ни контента, ни эмоций. По текущему состоянию **невозможно продать дорогой продукт** (7 800-9 200 ₽ — это импульсная покупка только при сильном бренде, которого пока нет).

Потенциал рынка огромен: Байкал зимой = миллионы туристов, рыбаки по всей Сибири, лыжники, ищущие альтернативу каткам. Бренд «ГРОМ» — **уникальный по нише** (собственное производство озёрных коньков в РФ практически отсутствует). С правильной подачей это легко может стать флагманским продуктом в категории «outdoor ice skating».

Главный приоритет: **наполнить товары смыслом** (описания, фото в деле, видео, отзывы) и **выстроить бренд-нарратив** (почему «ГРОМ», для кого, зачем). После этого — масштабирование через контент и рекламу.
