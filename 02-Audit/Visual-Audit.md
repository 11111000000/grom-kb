# 🎨 Визуальный аудит

> Дизайн, цвета, типографика, логотип, медиа

**Общая оценка: 4/10** — функционально, но без характера и брендинга.

---

## 🖼 Логотип

**Файл:** `gr-1.webp`, 200×182 px, 28 КБ, формат WebP, растровый

### Проблемы
- 🔴 **Растровый, не векторный** — на retina-экранах мылит
- 🔴 Нет **бренд-знака** — только текст «ГРОМ» в рамке
- 🔴 Нет версий для тёмного фона, favicon, печати
- 🟡 Нет monogram (иконки для favicon/соцсетей)

### Что нужно
- ✅ SVG-версия (master)
- ✅ Версия для светлого фона
- ✅ Версия для тёмного фона
- ✅ Monogram / фавикон
- ✅ Гайдлайн использования

## 🔤 Типографика

### Текущая
- **Шрифт:** Roboto (Google Fonts) + Elementor defaults
- **Иконки:** swiper-icons, wpshop-core

### Проблемы
- 🔴 Roboto — слишком нейтральный, нет характера
- 🔴 Нет акцидентного шрифта для заголовков
- 🔴 Нет кастомного шрифта для бренда
- 🟡 Шрифты загружаются с Google Fonts (GDPR/firewall concerns)

### Рекомендация

**UI / body:** Inter или Manrope (гротеск, читабельный)
**Заголовки:** TT Norms Pro или Onest (характерный, но читабельный)
**Акциденция / hero:** Бесплатный — Russo One или Unbounded (если нужен «громкий» голос)

**С учётом РФ-реалий:** лучше локально хостить, отказаться от Google Fonts.

## 📐 Композиция

### Hero (главная)
**Текущее:**
- Картинка gr.webp 1376×768 на всю ширину
- 2 кнопки поверх (через `bwd-dual-buttons`)
- Анимация `bounceInDown`

**Проблемы:**
- 🔴 Статичная картинка, нет эмоции
- 🔴 Кнопки поверх картинки — закрывают её
- 🔴 Нет подзаголовка, УТП

**Что должно быть:**
- Видео-фон (цикл 10-15 сек, лёд Байкала, катание)
- Overlay с заголовком + подзаголовком + CTA
- Липкий заголовок при скролле

## 🧩 UI-компоненты

### Кнопки
- **Стиль:** скруглённые, синие (стандарт темы)
- **Проблемы:** нет вариантов (primary, secondary, ghost, danger)
- **Нет состояний** (hover, focus, disabled)

### Карточки товаров
- **Стиль:** белые, скруглённые углы
- **Бейдж «Хит»** — на 3 товарах, обесценивается

### Иконки
- **Корзина:** wpshop-core (стандарт)
- **Избранное:** сердечко (стандарт)
