    🧭 UX-аудит¶

    Юзабилити, навигация, формы, поток покупки

    📊 Общая оценка UX: 3/10¶

    Сайт функционален, но имеет критические проблемы с навигацией и контентом, которые блокируют конверсию.

    🧭 Навигация¶

    Шапка¶

    Элементы: логотип · телефон · вход · корзина · избранное · гамбургер

    Проблемы: - 🔴 Меню из 2 пунктов (Магазин + Контакты) — фактически нет навигации - 🔴 Дублирование телефонов: шапка +7 (901) 641-48-48 ≠ контакты +7 (902) 576-19-17 - 🔴 Нет поиска по сайту - 🟡 Избранное и корзина — мелкие иконки, плохо различимы

    Что должно быть: - Каталог ▾ (с подкатегориями) - О бренде - Как выбрать размер - Доставка и оплата - Гарантия - Блог - Контакты

    Меню-категории (текущее)¶

    Только одна категория: «Гром» → 6 товаров в одной куче.

    Должно быть: - Готовые коньки (001, 002) - Лезвия (003, 004) - Крепления (006) - Аксессуары (005) - Кастом / Сделано на заказ

    🛒 Поток покупки (Customer Journey)¶

    Текущий флоу (плохой)¶

    Главная → "Перейти в магазин" → /магазин-спортивного-инвентаря/ 
  → клик по карточке → /product/<slug>/ 
  → "Выбрать параметры" → выбор размера → "В корзину" 
  → /cart/ → /checkout/

    Проблемы в флоу: 1. 🔴 Лишний клик: «Выбрать параметры» на variable-товарах вместо быстрого добавления 2. 🔴 Нет «В 1 клик» / быстрого заказа 3. 🔴 Самовывоз и доставка не очевидны 4. 🟡 Нет индикации «Товар в наличии / Под заказ / Нет в наличии»

    Идеальный флоу¶

    Hero-баннер → УТП-блок → Каталог хитов → Калькулятор размера → Карточка 
  → Описание + Видео + Отзывы → Sticky «В корзину» → Корзина → Checkout

    📄 Карточка товара¶

    Что есть¶

    • Галерея (Flexslider)
    • Заголовок
    • Цена
    • Пустое короткое описание
    • Выбор вариации (select)
    • Количество
    • Кнопки: «В корзину», избранное
    • Вкладки: Детали, Отзывы
    • Robokassa-виджет

    Что критически отсутствует¶

    • ❌ Описание (short_description пустое)
    • ❌ Характеристики (материал, сталь, заточка)
    • ❌ Видео
    • ❌ Отзывы (0 на всех товарах)
    • ❌ Cross-sell (с этим покупают)
    • ❌ Up-sell (похожие товары)
    • ❌ Таблица размеров
    • ❌ Sticky-кнопка при скролле

    UX-проблемы формы¶

    • При незаполненной вариации кнопка «В корзину» не даёт обратной связи
    • variation_id=0 в скрытом поле — невалидное состояние
    • Нет «Нашли дешевле? Сообщите»
    • Нет кнопки «Задать вопрос о товаре»

    📦 Каталог¶

    Текущий вид¶

    • 5 колонок на десктопе
    • Сортировка: Исходная / Популярность / Рейтинг / Новизна / Цена
    • Бейдж «Хит!» на 3 товарах

    Проблемы¶

    • 🔴 Нет фильтров (по размеру, цене, цвету)
    • 🔴 5 колонок = мелкие карточки
    • 🔴 Hover-эффект только на десктопе, на мобиле — нет
    • 🔴 Alt-ы фото = системные имена (IMG_20251128_155632_918)
    • 🟡 Бейдж «Хит» обесценивается (3 из 6)
    • 🟡 Нет quick-view (хотя кнопка есть в коде)

    🏠 Главная страница¶

    Что есть¶

    • Hero-картинка (gr.webp 1376×768)
    • Две кнопки: «Магазин спортивного инвентаря» / «Турагентство»
    • Больше ничего

    Что отсутствует (критично)¶

    • ❌ УТП (что за бренд, для кого)
    • ❌ Hero-CTA с ценностью (просто «куда-то ведёт»)
    • ❌ Блок преимуществ (почему мы)
    • ❌ Витрина хитов
    • ❌ Отзывы клиентов
    • ❌ Социальное доказательство (N довольных клиентов, фото)
    • ❌ FAQ
    • ❌ Форма подписки / лид-магнит
    • ❌ Видео-обзор

    SEO-проблема¶

    Главная — это просто картинка. Поисковик не понимает тематику сайта. OG-description: «озерные коньки гром озёрные коньки гром гром38» — спам keywords.

    🦶 Footer (подвал)¶

    Текущий¶

    • Копирайт «© 2025-2026 «ГРОМ» Все права защищены»
    • Ссылки: Политика конфиденциальности / Публичная оферта
    • Копирайт разработчика «Gruppa38»

    Проблемы¶

    • 🔴 Ссылка https:\\gruppa38.ru — обратный слэш (баг)
    • 🔴 Нет ссылок на соцсети
    • 🔴 Нет email-подписки
    • 🟡 Нет дублирующего меню

    🎯 Приоритетные UX-улучшения¶

    P0 (немедленно)¶

    1. Расширить меню до 6-7 пунктов
    2. Заполнить описания товаров
    3. Добавить категории в каталог
    4. Починить контактные данные (1 телефон, 1 email)

    P1 (до запуска)¶

    1. Калькулятор размера
    2. Sticky-кнопка «В корзину»
    3. Cross-sell на карточке
    4. Hero-блок с УТП

    P2 (рост)¶

    1. Поиск по сайту
    2. Фасетный фильтр
    3. Quick-view
    4. Wishlist-интеграция в email
