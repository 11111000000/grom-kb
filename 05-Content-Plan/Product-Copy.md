# 🛍 Описания товаров (Product Copy)

> Реальные описания + шаблоны для заполнения пустых

**Источник:** эпистемический аудит, оригинальные тексты сайта, база знаний проекта.

---

## 📋 Структура описания

Каждая карточка товара должна содержать:

1. **Заголовок** (H1) — название товара
2. **Лид-абзац** (50-80 слов) — суть продукта, главное УТП
3. **Блок «Для кого»** — целевая аудитория
4. **Блок «Где использовать»** — сценарии
5. **Характеристики** (таблица)
6. **Блок «Как использовать»** — инструкция
7. **Блок «Совместимость»** — с какими ботинками
8. **Блок «Уход»** — как точить, хранить
9. **FAQ** (3-5 вопросов)
10. **CTA** — купить, связаться

## 📝 Лезвие «ГРОМ» неокрашенное (SKU 003)

### Шаблон для заполнения (описание ПУСТОЕ)

**Лид-абзац (нужно написать):**
> Стальное лезвие для открытого льда ГРОМ. Длина 480 мм для ботинок 38-42, длина 520 мм для ботинок 43-46. Совместимо с лыжными ботинками NNN и SNS. Произведено в Ангарске, испытано на Байкале.

**Характеристики (известно):**

| Параметр | Значение |
|---|---|
| Материал | сталь (марка требует уточнения) |
| Длина | 480 мм / 520 мм |
| Вес | 1,1 кг |
| Габариты | 57 × 13 × 7 см |
| Артикул | 003 |
| Совместимость | NNN, SNS |
| Заточка | требует первичной обработки |

**Что нужно уточнить у производителя:**
- Марка стали (вероятно 420, как у готовых коньков)
- Толщина лезвия (вероятно 2 мм)
- Твёрдость (HRC)
- Радиус скругления

**Для кого:**
- Рыбаки-зимники с лыжными ботинками
- Лыжники, которые хотят кататься по открытому льду
- Outdoor-энтузиасты

**Где использовать:**
- Замёрзшие озёра
- Реки со льдом
- Байкал (январь-март)
- Горные озёра

**Как установить:**
1. Установите крепление на ботинок (если ещё нет)
2. Вставьте лезвие в крепление до щелчка
3. Проверьте фиксацию (потяните)
4. Сделайте 5 шагов по снегу для проверки
5. Готово — можно кататься

**FAQ:**
- **Q: Подойдёт ли под ботинок 42 размера?**
  A: Да, для 38-42 берите 480 мм, для 43-46 — 520 мм.
- **Q: Нужно ли точить перед первым катанием?**
  A: Да, фабричная заточка требует доработки под ваш стиль.
- **Q: Можно ли использовать для хоккея?**
  A: Нет, лезвие предназначено только для открытого льда.
- **Q: Какие ботинки подходят?**
  A: Любые лыжные ботинки с креплениями NNN или SNS.

**CTA:**
> Закажите сейчас — отправим СДЭК за 3-7 дней. По Ангарску — самовывоз.

## 📝 Озёрные коньки «ГРОМ» неокрашенные (SKU 001)

### РЕАЛЬНОЕ ОПИСАНИЕ (1217 символов, использовать как есть)

> «Нордики или как их ещё называют - байсы. Коньки для открытого льда собственной разработки и исполнения, предназначены под эксплуатацию с ботинками беговых лыж для конькового хода. Эти коньки результат долгой и кропотливой работы, как в теоретической части так и в практическом исполнении. В ходе испытаний была найдена и спроектирована оптимальная геометрия лезвия, скругление рабочей поверхности лезвия, оптимально эффективно распределена нагрузка передаваемая с крепления конька на сам конёк. Был найден компромисс толщины лезвия для оптимального сочетания управляемости и отдачи конька при манёврах и уверенным прохождением препятствий, с накатом и скольжением. В испытаниях принимали участие как профессиональные спортсмены, так и бывалые туристы с богатым опытом. После множества экспериментов модель была доведена до идеала, с точки зрения функционала и эргономики. Благодаря серийному производству, модель постоянно дорабатывается и улучшается. При разработке для сравнения тестировались различные финские и корейские модели, а так же некоторые наиболее заметные самоделки. Задача была сделать не просто какой-то конёк, а получить качественный спортивный инвентарь с отличными показателями и за умеренную цену.»

### Лид-абзац (на основе реального описания)
> Российские озёрные коньки ГРОМ собственной разработки. Сталь 420, толщина 2 мм, алюминиевый корпус. 109 отзывов с рейтингом 5.00. Под лыжные ботинки NNN и SNS. Испытаны на Байкале.

### Полные технические характеристики (из описания)

| Параметр | Значение |
|---|---|
| Цена | 7 800 ₽ |
| **Размеры** | **460 мм / 520 мм** |
| Вес пары без крепления | 800 г (460мм) / 880 г (520мм) |
| Вес с упаковкой | 1,3 кг |
| Габариты | 57 × 13 × 10 см |
| **Сталь лезвия** | 420 (нержавеющая) |
| **Толщина лезвия** | 2 мм |
| **Корпус** | алюминиевый уголок 20×20×2 мм |
| **Клёпки** | полнотелые алюминиевые |
| **Размер полки под крепление** | 42 мм |
| **Допустимый вес пользователя** | до 110 кг (есть усиленные) |
| **Радиус поворота** | 4 м (460мм) / 7 м (520мм) |
| **Ресурс заточки** | ≥200 км на чистом льду |
| **Производство** | лазерная резка на ЧПУ |
| **Совместимость** | NNN, SNS, Prolink |
| **Рейтинг** | 5.00 ⭐ (109 отзывов) |

### Как выбрать размер
- **Рост до 180 см** — обе модели комфортны, на 460 — больше манёвренности, на 520 — плавнее ход
- **Рост от 180 см и выше** — однозначно 520 мм, устойчивее
- **Большие расстояния** (50+ км) — 520 мм
- **Короткие прогулки, активный манёвр** — 460 мм

### Что входит в комплект
- Лезвие «ГРОМ» (неокрашенное)
- Крепление (требуется выбрать совместимое с вашими ботинками)

### Отзывы (выдержки)
- «откатал за сезон порядка 700км, не точил» — Антон
- «Коньки шикарные, сразу видно, что держишь в руках вещь» — Сергей
- «Очень лёгкие байсы, продавец порядочный, рекомендую» — Михаил

## 📝 Механическое крепление (SKU 006)

### РЕАЛЬНОЕ ОПИСАНИЕ (расшифровано из латиницы)

> «Несущие детали головной части крепления выполнены из металла, что несоизмеримо повышает его жизнеспособность и надёжность, по сравнению с креплениями выполненными полностью из пластика. Это крепление выполнено в виде монолитной пластины и не имеют отдельно крепящейся пятки, по этому оно очень хорошо подходит для эксплуатации на изготавливаемых нами коньках "байсов".»

### Характеристики

| Параметр | Значение |
|---|---|
| Цена | 1 200 ₽ |
| Вес | 0,35 кг |
| Габариты | 38 × 5 × 6 см |
| Тип | NNN, SNS |
| Материал несущих | металл |
| Артикул | 006 |

### Преимущества
- ✅ Металлические несущие (не пластик!)
- ✅ Монолитная пластина (без отдельно крепящейся пятки)
- ✅ Идеально для байсов ГРОМ
- ✅ Совместимость с NNN и SNS

### Что добавить
- Производитель крепления
- Конкретные модели совместимых ботинок (Salomon, Atomic, Fischer, Rossignol)
- Фото крепления на ботинке
