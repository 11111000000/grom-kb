## 📊 SWOT-матрица (обновлено)

### Стратегии SO (Сила + Возможности)
1. **S3+S4+O9:** Использовать 109 отзывов и полную Schema → SEO-лидерство в нише
2. **S5+S10+O10:** Технические характеристики + кейс 700 км → экспертный контент
3. **S8+O4:** Локальная поддержка + outdoor-сообщества → виральность

### Стратегии WO (Возможности - Слабости)
1. **O9+W1:** Использовать готовые отзывы для шаблонов описаний лезвий
2. **O10+W3:** Кейс 700 км на главную вместо пустой страницы
3. **O11+W14:** Лазерная резка ЧПУ → контент для блога
4. **O12+W13:** Гарантия 110 кг → добавить категорию «Для крупных пользователей»

### Стратегии ST (Сила + Угрозы)
1. **S1+S5+T3:** Собственное производство + тех. хар-ки → хеджирование цен на сталь
2. **S3+S4+T5:** Сильные отзывы и Schema → защита от копирования
3. **S6+S8+T4:** Прямые продажи + локальная поддержка → летние товары

### Стратегии WT (Слабости - Угрозы)
1. **W1+W11+T8:** Заполнить описания лезвий + унифицировать размеры + бэкап
2. **W3+W14+T5:** Наполнить главную + запустить блог → защита от копирования
3. **W5+W8:** robots.txt + meta description → технический долг
