## 🔴 КРИТИЧНЫЕ вопросы (нужны для решений)

### Бизнес
- [ ] **Какова реальная выручка ГРОМ за последние 1-2 года?**
  - Где взять: Яндекс.Метрика (ecommerce), данные владельца, бухгалтерия Т-банка
  - Без этого невозможно планировать

- [ ] **Сколько продаж в месяц/сезон?**
  - Где взять: админка WooCommerce
  - Без этого невозможно оценить конверсию

- [ ] **Кто основные клиенты? (рыбаки, туристы, лыжники?)**
  - Где взять: интервью с клиентами, анализ отзывов
  - Без этого невозможно позиционирование

- [ ] **Какие SKU продаются, а какие нет?**
  - Где взять: аналитика продаж
  - Без этого невозможно планировать ассортимент

### Маркетинг
- [ ] **Какой реальный трафик на сайт сейчас?**
  - Где взять: Яндекс.Метрика
  - Без этого невозможно оценить потенциал

- [ ] **Откуда приходят клиенты (SEO, прямые, сарафан)?**
  - Где взять: Яндекс.Метрика → источники
  - Без этого невозможно распределить бюджет

- [ ] **Какая доля мобильного трафика?**
  - Где взять: Яндекс.Метрика
  - Влияет на приоритет mobile-first
