## 🎯 Требования ГРОМ38 к хостингу

### Бизнес-требования
- ✅ Юрлицо в РФ (ООО «ГРОМ», ИНН 3801165352) — **нужен НДС, договор, ЭДО**
- ✅ 152-ФЗ (обрабатываем ПД клиентов: ФИО, адреса, телефоны, email)
- ✅ РобоКасса — должна работать (РФ-хостинг)
- ✅ Робот оплаты, СБП, эквайринг — без проблем
- ✅ Бизнес в РФ, аудитория в РФ
- ✅ Самозанятые/ИП/юрлица как клиенты (для туров)

### Технические требования
- ✅ Поддержка Node.js (для бэка)
- ✅ Поддержка Astro 5 / статики
- ✅ Поддержка Cloudflare Workers (если выберем этот путь)
- ✅ PostgreSQL или SQLite
- ✅ S3-совместимое хранилище (для фото)
- ✅ CDN в РФ (быстрая загрузка)
- ✅ SSL
- ✅ Мониторинг и алерты

### AI-агент требования
- ✅ CLI для управления
- ✅ API для интеграции
- ✅ Возможность деплоя Python-агента
- ✅ Webhook endpoints
- ✅ MCP-совместимость (желательно)
- ✅ GPU-тарифы (если будем self-hostить LLM)

## 🏆 ТОП-3 рекомендации для ГРОМ38

### 🥇 #1: Selectel (основной)

**Почему:**
- ✅ **Единственный с ФСТЭК №17 и №21** (если будем обрабатывать ПД с особыми категориями)
- ✅ **152-ФЗ УЗ-1** из коробки
- ✅ **Самый полный стек GPU** (T4, A100, RTX 4090) — для будущего LLM
- ✅ **Самый зрелый CLI** + SDK Go/Python
- ✅ **Managed K8s** — для оркестрации агентов
- ✅ **ML Platform** — готовая инфра для AI
- ✅ **3 зоны доступности** в МСК, СПб, НСК
- ✅ Включён в реестр РФ №9884
- ✅ Юрлицо/НДС/ЭДО — всё есть

**Тарифы для ГРОМ38:**

| Назначение | Тариф | Цена/мес |
|---|---|---|
| Бэкенд (Cloud VPS) | CL21-SSD (2 vCPU, 4 ГБ) | ~1 500 ₽ |
| БД (PostgreSQL) | Managed PostgreSQL 2 vCPU/4 ГБ | ~2 500 ₽ |
| S3 (фото, файлы) | Холодное хранение 100 ГБ | ~100 ₽ |
| **Итого на старте** | | **~4 000 ₽/мес** |
| + GPU (когда понадобится) | T4 16 ГБ | +27 900 ₽/мес |

**Команды для агента:**

```bash
# Установить CLI
pip install selvpcclient
# или
go install github.com/selectel/go-selvpcclient/v3/cmd/selvpc-cli@latest

# Создать сервер
selvpc-cli server create --name "grom-backend" --flavor "CL21-SSD"

# Создать PostgreSQL
selvpc-cli db create --type "postgres" --name "grom-db" --flavor "postgres-2-4"

# Создать S3
selvpc-cli s3 bucket create --name "grom-files"

# Деплой агента
kubectl apply -f grom-agent-deployment.yaml  # через Managed K8s
```

### 🥈 #2: Timeweb Cloud (если важен OpenClaw)

**Почему:**
- ✅ **OpenClaw** — встроенный оркестратор для AI-агентов (нет у других)
- ✅ MCP-интеграция из коробки
- ✅ 152-ФЗ УЗ-1 (но без ФСТЭК)
- ✅ Отличный UI, удобная панель
- ✅ Лучшая цена среди «большой пятёрки»
- ✅ **Проще** для старта (менее корпоративный)

**Тарифы для ГРОМ38:**

| Назначение | Тариф | Цена/мес |
|---|---|---|
| Бэкенд | Cloud MSK 50 (2 vCPU, 4 ГБ) | 1 062 ₽ |
| БД (на своём VPS) | PostgreSQL на том же сервере | 0 ₽ |
| S3 | Timeweb S3 | ~0.8 ₽/ГБ |
| **Итого на старте** | | **~1 100 ₽/мес** |
| + GPU (когда понадобится) | A100 (по запросу) | по запросу |

**Команды для агента (OpenClaw — уникальная фича):**

```bash
# Установить CLI
curl -fsSL https://timeweb.cloud/install.sh | bash

# Задеплоить AI-агента
twc agent deploy \
  --name "grom-agent" \
  --image "grom-agent:latest" \
  --mcp "chrome-devtools,emcp" \
  --env "ROBOKASSA_KEY=***,AIRTABLE_KEY=***" \
  --cpu 2 --memory 4

# Смотреть логи
twc agent logs grom-agent --follow
```

**Минусы:**
- ❌ GPU-тарифы непрозрачные (по запросу)
- ❌ Нет ФСТЭК (если понадобится обработка спецкатегорий ПД)
- ❌ Меньше managed-сервисов, чем у Selectel

### 🥉 #3: Yandex Cloud (если экосистема)

**Почему:**
- ✅ Готовый LLM через **Inferno** (Qwen, Llama) — pay-per-token
- ✅ S3, DBaaS, CDN из коробки
- ✅ Самый мощный CLI (`yc` с gRPC)
- ✅ Yandex Monitoring (managed Prometheus)
- ✅ Yandex Lockbox для секретов

**Когда выбирать:**
- Если хотите **готовый LLM** без своего GPU
- Если нужна **интеграция с Яндекс.Метрикой** (уже используется)
- Если используете **OAuth через Яндекс**

**Минусы:**
- ❌ Привязка к экосистеме Яндекса
- ❌ Нет ФСТЭК для обычных ПД
- ❌ Yandex Cloud может блокировать за рубежом

## 💎 Моя рекомендация для ГРОМ38

### Старт: **Selectel** (основной)

**Архитектура:**

```
┌─────────────────────────────────────────────┐
│           Selectel (МСК)                    │
│                                             │
│  ┌────────────────────────────────────┐    │
│  │  Cloud VPS (CL21-SSD)             │    │
│  │  - Astro 5 (статика)              │    │
│  │  - Cloudflare Tunnel              │    │
│  │  - Hono/Fastify (API бэк)         │    │
│  │  - Python AI-агент                │    │
│  │  - Nginx reverse proxy            │    │
│  │  - Certbot SSL                    │    │
│  └────────────────────────────────────┘    │
│                  │                          │
│  ┌──────────────┼──────────────┐           │
│  │              │              │           │
│  ▼              ▼              ▼           │
│ PostgreSQL  S3 (R2)     Cloudflare CDN     │
│ Managed     100 ГБ      (кеш РФ)           │
│ 4 ГБ RAM    100 ₽/мес   бесплатно          │
│ 2 500 ₽                                    
└─────────────────────────────────────────────┘
```

**Стоимость на старте:**

| Компонент | Цена/мес |
|---|---|
| Cloud VPS CL21-SSD (2 vCPU, 4 ГБ) | 1 500 ₽ |
| Managed PostgreSQL (2 vCPU, 4 ГБ) | 2 500 ₽ |
| S3 холодное (100 ГБ) | 100 ₽ |
| **Итого** | **~4 100 ₽/мес** |
| Домен гром38.рф | 1 000 ₽/год |

**Что делаем с этим:**

1. **Астро 5** — статический фронт на Cloudflare Pages (бесплатно)
2. **Cloudflare Tunnel** — проксирование к VPS Selectel (безопасность)
3. **Hono** — лёгкий API бэк на VPS
4. **Python AI-агент** — в Docker на том же VPS
5. **Managed PostgreSQL** — для заказов, бронирований, отзывов
6. **S3** — для фото и видео товаров
7. **Cloudflare Pages** — CDN для статики (кеш в РФ)
8. **R2 (Cloudflare)** — для бэкапов (10 ГБ бесплатно)

**Преимущества:**
- ✅ Один провайдер (Selectel) — один биллинг, одна поддержка
- ✅ Cloudflare Pages + Workers — для статики и edge
- ✅ Managed PostgreSQL — не нужно администрировать БД
- ✅ 152-ФЗ из коробки
- ✅ Когда понадобится LLM — T4/A100 готовые конфиги
- ✅ Когда понадобится K8s — Managed K8s

## 🔧 Альтернативный сценарий: гибрид

**Если хочется Cloudflare Workers (edge):**

```
┌─────────────────────────────────────────────┐
│         Cloudflare Pages + Workers           │
│  (статика + edge API, бесплатно)            │
└──────────────────┬──────────────────────────┘
                   │
                   ▼
┌─────────────────────────────────────────────┐
│         Selectel Managed PG                 │
│  (заказы, бронирования, отзывы)            │
│  2 500 ₽/мес                                │
└─────────────────────────────────────────────┘
```

**Стоимость:** ~2 500 ₽/мес (вместо 4 100 ₽)
**Но:** меньше контроля, нужны Edge-функции вместо полного бэка
