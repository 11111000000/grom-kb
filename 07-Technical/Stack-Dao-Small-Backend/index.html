    ☯️ Дао маленького бэка¶

    «Когда мудрец управляет людьми, он ставит себя ниже них. Тогда они чувствуют себя свободными.»

    🌱 Ответ: Дао допускает¶

    Маленький свой бэк — это не нарушение Дао. Это его продолжение.

    Но Дао маленького бэка отличается от Дао большого:

    Большой бэк (анти-Дао) Маленький бэк (Дао)
    Заменяет внешние сервисы Дополняет их
    Делает всё сам Делает только то, что не могут другие
    Требует DevOps Требует 1 разработчика
    100% контроль 80% контроля достаточно
    Хостится на Railway/AWS Хостится на Cloudflare/Vercel
    1000+ строк кода 100-200 строк
    Горизонтальное масштабирование Вертикальное (больше памяти)
    PostgreSQL на своём сервере SQLite / D1 / Turso

    🎯 Что бэк ДЕЛАЕТ (Дао-принцип)¶

    Бэк делает только то, что невозможно сделать статикой + 2-3 внешними сервисами.

    ✅ Что бэк ДОЛЖЕН делать¶

    1. AI-агент endpoint — /api/agent/chat или /mcp
    2. Webhook receiver — от РобоКассы, Telegram, Calendly
    3. Калькулятор размера — логика подбора (если сложнее)
    4. Email-уведомления — если не хотим Resend
    5. Аналитика событий — сбор поведения
    6. Бронирования — если нужна сложная логика (календарь + места)

    ❌ Что бэк НЕ ДОЛЖЕН делать¶

    • ❌ Каталог товаров — Airtable (бесплатно, проще)
    • ❌ Каталог туров — Airtable или Notion
    • ❌ Хранение файлов — Cloudflare R2 или Supabase Storage
    • ❌ Аутентификация — Clerk или Supabase Auth
    • ❌ База данных — Supabase (если нужна) или SQLite
    • ❌ Админка — Airtable интерфейс или Refine

    🌿 Три варианта маленького бэка¶

    🥉 Вариант 1: «Бамбук» — Cloudflare Workers + D1¶

    Самый гармоничный. 0-5 $/мес.

    Компонент Технология
    Бэк Cloudflare Workers (V8 isolates)
    БД D1 (SQLite на edge)
    Файлы R2 (S3-совместимое, 10 ГБ бесплатно)
    Фронт Astro 5 (статика на CF Pages)
    AI-агент Cloudflare Workers AI (встроенные LLM)
    Vector DB Vectorize (встроенный векторный поиск)
    Email Resend (бесплатно 3000/мес)
    Каталог Airtable (через API)
    Туры Airtable или D1
    Бронирования D1 + простая логика
    Хостинг Cloudflare Pages + Workers (всё в одном!)

    Стоимость: 0-5 $/мес

    Плюсы: - ✅ Всё в одном месте (Cloudflare) - ✅ Edge по всему миру - ✅ 100 000 запросов/день бесплатно - ✅ D1 — настоящая SQL БД - ✅ Workers AI — LLM встроены - ✅ Vectorize — семантический поиск - ✅ R2 — файлы - ✅ TypeScript everywhere - ✅ Один биллинг

    Минусы: - 🟡 Workers — только V8 (не Node.js) - 🟡 D1 — SQLite (не для огромных данных) - 🟡 Нет PostgreSQL (но для ГРОМ не нужно)

    Идеально для: магазина + турагентства до 100 000 товаров/туров

    🥈 Вариант 2: «Ива» — Hono на Vercel/Cloudflare + Supabase¶

    Гибкий. 0-25 $/мес.

    Компонент Технология
    Бэк Hono (легковесный, edge-ready)
    Фронт Astro 5 (статика)
    БД Supabase (PostgreSQL)
    Файлы Supabase Storage
    Auth Supabase Auth
    Realtime Supabase Realtime (WebSocket)
    Vector pgvector (встроен в Supabase)
    AI-агент OpenAI/Anthropic API + MCP
    Email Resend
    Каталог Supabase (одна БД)
    Туры Supabase (одна БД)
    Хостинг Vercel (фронт + API) или Cloudflare Pages

    Стоимость: 0-25 $/мес

    Плюсы: - ✅ Hono — 14 КБ, edge-ready - ✅ Supabase — полная платформа - ✅ PostgreSQL — надёжная БД - ✅ Real-time из коробки - ✅ Auth из коробки - ✅ Можно мигрировать на свой сервер

    Минусы: - 🟡 2 провайдера (Vercel + Supabase) - 🟡 Supabase в РФ может быть медленным

    Идеально для: если нужна PostgreSQL и real-time

    🥇 Вариант 3: «Сосна» — Fastify на Vercel + Prisma + PostgreSQL¶

    Мощный, классический. 0-30 $/мес.

    Компонент Технология
    Бэк Fastify или Hono (Node.js)
    ORM Prisma или Drizzle
    Фронт Astro 5
    БД Neon или Supabase (PostgreSQL serverless)
    Файлы Cloudflare R2
    AI-агент OpenAI/Anthropic + MCP
    Email Resend
    Каталог PostgreSQL
    Туры PostgreSQL
    Хостинг бэка Vercel Functions или Railway
    Хостинг фронта Vercel или Cloudflare Pages

    Стоимость: 0-30 $/мес

    Плюсы: - ✅ Fastify — один из самых быстрых - ✅ Prisma — лучший ORM - ✅ Neon — serverless PostgreSQL с branching - ✅ Полный контроль

    Минусы: - 🟡 Больше кода - 🟡 Нужно настраивать больше

    Идеально для: сложной бизнес-логики

    💎 Мой Дао-выбор (маленький бэк)¶

    Начинаем с «Бамбука» (Cloudflare Workers + D1)¶

    Почему:

    1. Дао простоты:
    2. 1 провайдер (Cloudflare)
    3. 1 биллинг
    4. 1 стек (TypeScript)

    5. 1 дашборд

    6. Дао экономии:

    7. 0-5 $/мес на старте
    8. 100 000 запросов/день бесплатно

    9. Масштабируется только при росте

    10. Дао скорости:

    11. Edge по всему миру
    12. <50 мс холодный старт

    13. D1 — быстрая для чтения

    14. Дао надёжности:

    15. Cloudflare SLA 100%
    16. 300+ точек присутствия

    17. Автоматическое масштабирование

    18. Дао AI:

    19. Workers AI встроены
    20. Vectorize для семантического поиска
    21. MCP через HTTP

    Что где хранится¶

    Данные Где Почему
    Товары (каталог) Airtable Просто управлять, API
    Туры (каталог) Airtable Календарь, проще
    Блог (статьи) Markdown в репо Версионирование
    Заказы D1 Нужна SQL-логика
    Бронирования D1 Нужна SQL-логика
    Отзывы D1 Быстрый доступ
    Файлы (фото, видео) R2 Дёшево, 10 ГБ бесплатно
    Кэш KV Быстро, edge
    Сессии KV TTL

    AI-агент (простой)¶

    // worker/src/ai/agent.ts
import { Ai } from '@cloudflare/ai';

export async function agent(messages: Message[]) {
  const ai = new Ai(env.AI);
  const response = await ai.run('@cf/meta/llama-3.1-8b-instruct', {
    messages: [
      { role: 'system', content: 'Ты — AI-продавец ГРОМ...' },
      ...messages
    ],
    tools: [
      {
        name: 'search_products',
        description: 'Поиск товаров в каталоге',
        parameters: { query: 'string' }
      },
      {
        name: 'create_order',
        description: 'Создать заказ',
        parameters: { product_id: 'string', size: 'string' }
      }
    ]
  });
  return response;
}

    LLM бесплатно через Workers AI (10 000 запросов/день).

    🆚 Миграция с «Бамбука» на «Сосну»¶

    Если ГРОМ вырастет:

    Бамбук (старт) → Ива (рост) → Сосна (масштаб)
0-5 $/мес        0-25 $/мес    0-30 $/мес
Cloudflare       Hono+Supabase  Fastify+Prisma
D1 (SQLite)      PostgreSQL     PostgreSQL
Workers AI       OpenAI         OpenAI

    Миграция простая: - Код бэка — почти без изменений - БД — экспорт D1 → PostgreSQL - Фронт — без изменений - AI — без изменений
