# 🔍 SEO-техника

> Техническое SEO для сайта

---

## 📋 Чек-лист технического SEO

### On-page

#### Title
- [ ] Уникальный на каждой странице
- [ ] 50-60 символов
- [ ] Содержит ключевое слово
- [ ] Бренд в конце

#### Description
- [ ] Уникальный на каждой странице
- [ ] 150-160 символов
- [ ] Содержит ключевые слова
- [ ] CTA в конце

#### Заголовки
- [ ] H1 — один на странице
- [ ] H2-H3 — логическая иерархия
- [ ] Ключевые слова в H1

#### Изображения
- [ ] Alt-ы осмысленные (не «IMG_123»)
- [ ] WebP/AVIF формат
- [ ] Сжатие < 100 КБ
- [ ] Lazy-loading

#### URL
- [ ] Короткие, читаемые
- [ ] Транслит (или кириллица для Яндекса)
- [ ] Без параметров (?p=123)
- [ ] ЧПУ включён

#### Контент
- [ ] Уникальный (не копипаст)
- [ ] 300+ слов на странице
- [ ] Внутренняя перелинковка
- [ ] LSI-слова

### Technical

#### Robots
- [ ] `robots.txt` настроен
- [ ] `<meta name="robots">` корректный
- [ ] Страницы в разработке — `noindex`

#### Sitemap
- [ ] XML sitemap создан
- [ ] Зарегистрирован в Google Search Console
- [ ] Зарегистрирован в Яндекс.Вебмастер
- [ ] Обновляется автоматически

#### Schema.org
- [ ] Organization
- [ ] Product
- [ ] BreadcrumbList
- [ ] LocalBusiness
- [ ] FAQPage (где есть FAQ)

#### Скорость
- [ ] PageSpeed > 80
- [ ] TTFB < 800 мс
- [ ] LCP < 2.5 сек

#### Мобильная версия
- [ ] Mobile-friendly
- [ ] Core Web Vitals зелёные
- [ ] Touch-элементы > 48px

#### HTTPS
- [ ] SSL-сертификат установлен
- [ ] Все страницы на HTTPS
- [ ] 301-редирект с HTTP на HTTPS

#### Canonical
- [ ] `<link rel="canonical">` на каждой странице
- [ ] Указывает на предпочтительный URL

#### Hreflang
- [ ] Не нужен (только русский)

## 📄 Шаблоны мета-тегов

### Главная
```html
<title>ГРОМ — озёрные коньки для открытого льда | Сделано в Ангарске</title>
<meta name="description" content="Российский производитель озёрных коньков (байсов) из Ангарска. Лезвия и готовые коньки для лыжных ботинок NNN/SNS. Испытано на Байкале. Доставка по РФ. Гарантия 1 год.">
```

### Категория «Готовые коньки»
```html
<title>Готовые озёрные коньки ГРОМ — купить от 7 800 ₽ | гром38.рф</title>
<meta name="description" content="Готовые озёрные коньки ГРОМ в наличии. Окрашенные и неокрашенные байсы для лыжных ботинок NNN и SNS. Гарантия 1 год. Доставка по РФ за 3-7 дней.">
```

### Категория «Лезвия»
```html
<title>Лезвия для озёрных коньков ГРОМ — купить от 6 600 ₽ | гром38.рф</title>
<meta name="description" content="Стальные лезвия для озёрных коньков ГРОМ. Размеры 480 и 520 мм. Под лыжные ботинки NNN и SNS. Сделано в Ангарске. Испытано на Байкале.">
```

### Карточка товара
```html
<title>Лезвие «ГРОМ» неокрашенное — 6 600 ₽ | Купить в гром38.рф</title>
<meta name="description" content="Стальное лезвие для открытого льда. Длина 480 мм. Под лыжные ботинки NNN/SNS. Сделано в Ангарске. Гарантия 1 год. Доставка 3-7 дней по РФ.">
```

### Блог (статья)
```html
<title>Как выбрать озёрные коньки: гайд для новичка | ГРОМ</title>
<meta name="description" content="Подробный гайд по выбору озёрных коньков: размер, тип крепления, материал. Советы от производителя байсов ГРОМ.">
```

## 🗺 Sitemap

### Структура
```
гром38.рф/sitemap.xml
├── sitemap-products.xml (товары)
├── sitemap-categories.xml (категории)
├── sitemap-pages.xml (страницы)
└── sitemap-posts.xml (блог)
```

### Приоритеты
```
Главная: 1.0
Категории: 0.9
Товары: 0.8
Блог: 0.7
Контакты: 0.5
```

## 🔗 Внутренняя перелинковка

### Схема
```
Главная → Категории → Товары → Блог
   ↓          ↓          ↓
  Бренд   Сравнения   Гайды
```

### Примеры перелинковки
- Главная → ссылка на хиты
- Категория → ссылка на товары
- Товар → ссылка на «Похожие товары»
- Товар → ссылка на «Как выбрать» (блог)
- Блог → ссылка на товары

### Анкоры
- Точные ключи
- Разбавленные (брендовые, общие)
- Без спама
