# ADR-003: Брендинг — новый логотип

**Статус:** в обсуждении
**Дата:** 26.06.2026

---

## 📋 Контекст

Текущий логотип — растровое изображение `gr-1.webp` 200×182, без бренд-знака. Нужно решить, делать ли ребрендинг.

---

## 🔍 Варианты

### Вариант A: Оставить текущий

**Плюсы:**
- ✅ Бесплатно
- ✅ Быстро
- ✅ Узнаваемость (если есть)

**Минусы:**
- ❌ Растровый (мылит на retina)
- ❌ Нет бренд-знака
- ❌ Нет версий для разных фонов
- ❌ Не отражает суть бренда

### Вариант B: Доработать текущий

**Сделать:**
- Перевести в SVG
- Добавить бренд-знак (молния/лезвие)
- Создать версии для разных фонов

**Плюсы:**
- ✅ Дешевле (5-15 тыс. ₽)
- ✅ Сохраняем преемственность

**Минусы:**
- ❌ Старый логотип не отражает бренд-платформу
- ❌ Сложно улучшить без переделки

### Вариант C: Полный ребрендинг

**Сделать:**
- Новый логотип с нуля
- Брендбук (гайдлайн использования)
- Все версии (primary, mono, white, black, favicon)

**Плюсы:**
- ✅ Современный, продуманный
- ✅ Отражает бренд-платформу
- ✅ Работает на всех носителях
- ✅ Запоминается

**Минусы:**
- ❌ Дороже (30-100 тыс. ₽)
- ❌ Дольше (2-4 недели)
- ❌ Нужно заново привыкать

---

## 🎯 Решение (рекомендация)

**Вариант C — полный ребрендинг.**

### Обоснование
1. **Бренд-платформа требует визуального воплощения** — иначе это слова
2. **Конкуренты имеют сильные бренды** — нужно отстраиваться
3. **Логотип будет работать 5+ лет** — инвестиция окупается
4. **Повышает доверие** — серьёзный бренд выглядит дороже
5. **На всех носителях** — упаковка, визитки, соцсети

### Концепция логотипа

**Идея:** Молния + лезвие конька + текст «ГРОМ»

**Варианты:**

#### Вариант 1: «Молния-лезвие»
```
[⚡]  ГРОМ
     байсы · Ангарск
```

#### Вариант 2: «Стилизованное лезвие»
```
ГРОМ
[━ геометрия лезвия ━]
```

#### Вариант 3: «Монограмма»
```
[G⚡] или [Г⚡]
```

### Что должно быть в брендбуке

- Логотип primary (горизонтальный)
- Логотип вертикальный
- Бренд-знак (только иконка)
- Монохромная версия
- Белая версия
- Версия для тёмного фона
- Favicon
- Минимальный размер
- Охранное поле
- Недопустимое использование
- Цветовые коды (Pantone, CMYK, RGB, HEX)
- Шрифтовые пары
